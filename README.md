# Feitengpai-Network-Cable-Quality-Sampling-and-Dataset-Tool
This project targets network/cable testing for electronics competitions. A computer controls Feitengpai to sample Ethernet cables, logging link status, negotiated speed, ping, iperf3, NIC errors, and raw outputs. Python scripts and CSVs manage reproducible, labeled, and GitHub-ready workflows, including faulty or shorted cables.
